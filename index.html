SNESology Demo

This is an early demo of the SNESology project's free online music composer called Audio Source Composer. Follow our Facebook for release updates. Visit our Discord and tell us what you think. Check back soon for updates!

Project Introduction

The SNESology Project allows anyone with a web browser to compose and share original music based on the Super Nintendo Entertainment System Sound chip, and other consoles too!

Project Goals

  • Create music from any browser. No software to install.
  • Songs built with web technology. No sample libraries to manage.
  • Edit songs in real-time with your friends.
  • Framework for professional, collaborative, & administrative services.
  • Chip support: C64, NES, SNES, Sega, PSx, etc.
  • Open source & free forever

Want to contribute?

If you have an idea for the project, skill to lend, or you think you can contribute in some other way: We're looking for programmers, graphic artists, musicians, beta testers, and wishlists. Join our Discord and check out our github README for information on how to help.
